# Dynamic Background Color Control

## Overview

Experiment with background colors in real-time! The Color System panel now has **prominent background controls** to quickly try different looks.

---

## What's Available

### 🎨 Background Color Picker
**Large, prominent color input** at the top of the Color System panel:
- Live color picker (click to open system color chooser)
- Displays current hex code
- Shows real-time preview in viewport
- Changes apply instantly

### Quick Background Buttons
**8 preset dark backgrounds** for instant testing:
1. **Black** (#000000) - Pure black, maximum drama
2. **Navy** (#0f172a) - Current default, dark blue
3. **Purple** (#1e1b4b) - Deep purple, mysterious
4. **Brown** (#1a1410) - Warm, earthy brown
5. **Dark** (#0d1117) - GitHub dark theme
6. **Forest** (#0f2f1f) - Deep forest green
7. **Deep** (#1a0033) - Dark magenta
8. **Gray** (#111827) - Neutral dark gray

---

## How to Use

### Method 1: Quick Click (Fastest)
1. Open **Color System panel** (top right)
2. Scroll down to **Quick Backgrounds**
3. Click any color button
4. Watch the hero background change instantly 🎨

### Method 2: Custom Color Picker
1. Open **Color System panel**
2. Click the large color square next to "Background Color"
3. Choose any color
4. Close the picker
5. Background updates in real-time

### Method 3: Type Hex Code
If you know the hex code:
1. Open **Color System panel**
2. Look for the hex value display
3. Use a hex converter/picker
4. Click and enter your hex code

## Workflow

### Step 1: Find Your Look
1. Click through the quick background buttons
2. Find the vibe you like
3. Take note of the hex code

### Step 2: Fine-Tune (Optional)
1. Click the color picker for more control
2. Adjust brightness/saturation
3. Get the exact color you want

### Step 3: Lock It In
1. When satisfied, click 🔒 **Lock** (top right)
2. Click ⬇️ **Export** to save config
3. The color is saved in your `hero-config.json`

## Design Tips

### Complementary Colors
The 30/60/10 system automatically adjusts:
- **Dominant** (background you choose)
- **Secondary** (supporting color adjusts)
- **Accent** (highlights stay vibrant)

### Testing Different Backgrounds
Try at least 3-4 options:
1. One very dark (#000000)
2. One medium dark (#0f172a)
3. One with color tint (#1a1410 brown or #0f2f1f green)
4. One your brand color

See which feels right!

### Accessibility
Remember to check:
- Is the text readable?
- Is the hero section visible?
- Do colors feel professional?
