# Process — Bryan Balli · hitloop.agency

Phased, approval-gated workflow. Each phase requires sign-off before the next begins — no scope creep, no runaway builds.

1. **Discovery** — Async intake brief, competitive review, technical audit of existing stack.
2. **Scope** — Deliverable map and effort estimate. Approved before any design or code begins.
3. **Prototype** — Working prototype of key interactions. Approved before full build.
4. **Build** — Production implementation against approved prototype and spec.
5. **Handoff** — Documentation, deployment, and knowledge transfer.
